Komiku REST API Documentation

API untuk mengakses data manhwa dari Komiku. Klik tombol copy untuk menyalin URL endpoint.

Base URL: http://localhost:3000/api

GET Get All Manhwa

Mendapatkan daftar semua manhwa dengan pagination

GET http://localhost:3000/api/manhwa?page=1&limit=20

Query Parameters:

  • page (optional) - Halaman yang ingin diakses (default: 1)
  • limit (optional) - Jumlah data per halaman (default: 20)

GET Get Latest Manhwa

Mendapatkan manhwa dengan update terbaru

GET http://localhost:3000/api/manhwa/latest?limit=10

Query Parameters:

  • limit (optional) - Jumlah data yang ingin ditampilkan (default: 10)

GET Get Manhwa Detail

Mendapatkan detail manhwa berdasarkan ID

GET http://localhost:3000/api/manhwa/:id

Path Parameters:

  • id (required) - ID atau title manhwa

GET Get Chapters by Manhwa

Mendapatkan semua chapter dari manhwa tertentu

GET http://localhost:3000/api/chapters/:manhwaId?page=1&limit=50

Parameters:

  • manhwaId (required) - ID manhwa
  • page (optional) - Halaman (default: 1)
  • limit (optional) - Jumlah per halaman (default: 50)

GET Search Manhwa

Mencari manhwa berdasarkan judul atau genre

GET http://localhost:3000/api/search?q=keyword

Query Parameters:

  • q (required) - Kata kunci pencarian

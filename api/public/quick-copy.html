BASE URL
http://localhost:3000/api
GET ALL MANHWA
http://localhost:3000/api/manhwa?page=1&limit=20
GET LATEST MANHWA
http://localhost:3000/api/manhwa/latest?limit=10
GET MANHWA DETAIL
http://localhost:3000/api/manhwa/:id
GET CHAPTERS
http://localhost:3000/api/chapters/:manhwaId?page=1&limit=50
GET CHAPTER DETAIL
http://localhost:3000/api/chapters/:manhwaId/:chapterId
SEARCH MANHWA
http://localhost:3000/api/search?q=keyword
